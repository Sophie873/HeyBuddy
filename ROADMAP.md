# 개발 로드맵

## 완료
- 기본 음성 입력/출력 + CLI 에이전트 호출 파이프라인
- 상태 로그, 비프 피드백, 취소·종료 키워드
- 오프라인 STT 전환 경로(`vosk`) 추가
- 웨이크 키워드(옵션) 지원
- 대화 세션 요약/스크롤 저장 (`sessions/` 디렉터리, JSONL, 세션별 자동 저장)
- 슬래시 명령어 시스템 (`/기록`, `/지난기록`, `/도움`)
- `--history` CLI 플래그로 지난 세션 조회

## 다음 단계
1. 사용자 음성 볼륨 기반 자동 게인 자동조정
2. 스트리밍 STT/중간 결과 표시를 통한 인식 지연 체감 개선
3. `wake_word`로 시작하지 않을 때는 자동 무시하거나, 항상 처리 모드 토글 제공
